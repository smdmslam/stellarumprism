Do this:

- `.terminal-host` stays the outer shell
- add an inner `.terminal-stage` that is the **actual xterm mount node**
- make `.terminal-stage` narrower with an explicit right gap
- open xterm into `.terminal-stage`
- observe/fit against that exact element

That gives xterm a **truly smaller layout box**, so wrapping happens earlier by construction.

### Why this is better

- `margin-right: 24px` on `.terminal-stage` creates a **real narrower box**
- FitAddon measures the node xterm is actually mounted into
- text wraps before the scrollbar because xterm literally has less width
- no fake “column shaving”
- no dependence on `scrollbar-gutter` being honest on macOS

```ts
private fitTerminal(): void {
  if (!this.fit || !this.term) return;
  const host = this.root.querySelector<HTMLDivElement>(".terminal-stage");
  if (!host) return;
  if (host.clientWidth <= 0 || host.clientHeight <= 0) return;
  requestAnimationFrame(() => {
    try {
      this.fit.fit();
    } catch {
      // best-effort
    }
  });
}
```
