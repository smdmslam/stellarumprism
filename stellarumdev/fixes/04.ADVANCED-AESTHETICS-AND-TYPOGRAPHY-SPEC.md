# Advanced Aesthetics & Typography Specification (Cyber-Noir Evolution)

## 1. Architectural Intent & Design Philosophy
This document details the remaining aesthetic, structural, and typography enhancements designed to elevate Prism from a functional developer tool into a premium, state-of-the-art "Cyber-Noir" workspace.

### Core Pillars
1. **Premium Monospace / Editorial Hierarchy:** Crisp typography scaling across both rendered HTML prose and CodeMirror buffers.
2. **Surface Elevation & Depth:** Glassmorphism layers, top-edge specular highlights, and ambient drop shadows.
3. **Semantic Micro-badging:** Upgrading inline tokens into polished UI pills.

### 2.1 Code Block Elevation
Currently, code blocks in prose and agent tools are flat containers. We will upgrade them to elevated surfaces:
- **Top-Edge Highlight:** `border-top: 1px solid rgba(255, 255, 255, 0.12);` to simulate overhead specular lighting.
- **Ambient Shadow:** `box-shadow: 0 8px 24px rgba(0, 0, 0, 0.4);`
- **Floating Language Badge:** Absolute-positioned pill at the top-right of code blocks displaying the language (`ts`, `rust`, `css`) in dimmed cyan (`color: #38bdf8; background: rgba(56, 189, 248, 0.1);`).

### 2.2 Rich Admonition Callouts
Convert standard markdown blockquotes (`> [!NOTE]`, `> [!IMPORTANT]`) into rich UI alert boxes:
- **Accent Gutter:** 3px left border (`border-left: 3px solid #38bdf8;`).
- **Tinted Fill:** `background: rgba(56, 189, 248, 0.08);`.
- **Distinct Iconography / Headers:** Uppercase tracked headers (`font-weight: 700; letter-spacing: 0.05em;`).

### 2.3 CodeMirror Advanced Typography Tracking
In `src/file-editor.ts`, further tighten the heading hierarchy:
- **H1 Tracking:** `letterSpacing: "-0.02em"`, `lineHeight: "1.3"`.
- **H2 Tracking:** `letterSpacing: "-0.01em"`, `lineHeight: "1.35"`.
- **Semantic String Override:** Ensure specialized string tags (`t.special(t.string)`) cleanly override underlying default theme colors.

### 2.4 Slate-Indigo / Midnight Palette
Refine ambient backgrounds across panels:
- **Center Pane:** `#0b0f17`
- **Agent Pane:** `#0d111a`
- **Micro-transitions:** `transition: background-color 150ms ease, border-color 150ms ease;` on interactive surface elements.
