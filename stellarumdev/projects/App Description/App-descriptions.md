This project, Prism, is an AI-powered development environment designed to provide reliable coding assistance. Unlike typical AI coding tools, it prioritizes deterministic checks through various "substrate cells" (like typechecking, AST resolution, and test runners) before presenting findings to the user. This approach aims to guarantee correctness by only graduating findings to higher confidence levels when backed by verifiable evidence from these checks, resulting in a more trustworthy and less error-prone development experience.
