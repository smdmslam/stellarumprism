# WARP-Like File Viewer Enhancement
## Implementation Notes - April 29, 2026

## Overview
Goal: Transform Prism's plain text file viewer into a WARP-like rich editor with:
1. Syntax highlighting for all common file types
2. Intelligent word wrapping (not character-based)
3. Markdown preview within the editor
4. Improved typography and readability
5. All while maintaining full editability

## Current State Analysis
- File viewer uses CodeMirror (`@codemirror/*` packages)

#### B. Language Detection Function
Add a function to detect file type and return appropriate language extension:

```typescript
function getLanguageExtension(filePath: string) {
  const ext = filePath.toLowerCase();
  if (ext.endsWith('.md') || ext.endsWith('.markdown')) {
    return markdown({ codeLanguages: languages });
  } else if (ext.endsWith('.js') || ext.endsWith('.jsx')) {
    return javascript({ jsx: true });
  } else if (ext.endsWith('.ts') || ext.endsWith('.tsx')) {
    return javascript({ jsx: true, typescript: true });
  } else if (ext.endsWith('.json')) {
    return json();
  } else if (ext.endsWith('.py')) {
    return python();
  } else if (ext.endsWith('.html')) {
    return html();
  } else if (ext.endsWith('.css')) {
    return css();
  } else if (ext.endsWith('.xml') || ext.endsWith('.yml') || ext.endsWith('.yaml')) {
    return yaml();
  } else if (ext.endsWith('.sql')) {
    return sql();
  }
  return null; // Use plain text
}
```
