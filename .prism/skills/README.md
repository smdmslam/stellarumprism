# Prism Skills

This folder contains reusable Prism skills: durable guidance files that help the agent perform recurring tasks with better structure, framing, and consistency.

## VC pitch deck skill family

The `vc-pitchdeck-` files are a grouped companion set for creating, rewriting, reviewing, and presenting investor pitch decks.

### Files
- `vc-pitchdeck-index.md` — overview and recommended order of use
- `vc-pitchdeck-framing.md` — core strategic reframing from product description to investor value
- `vc-pitchdeck-framing-example-prism.md` — Prism-specific case study of a successful rewrite
- `vc-pitchdeck-workflow-checklist.md` — practical end-to-end process checklist
- `vc-pitchdeck-investor-language.md` — stronger investor-facing wording and phrasing patterns
- `vc-pitchdeck-slide-patterns.md` — reusable slide formulas by deck section
- `vc-pitchdeck-redflags.md` — common failure modes and how to catch them
- `vc-pitchdeck-speaker-notes.md` — how to turn sparse slides into a strong spoken pitch

### Recommended order
1. `vc-pitchdeck-index.md`
2. `vc-pitchdeck-workflow-checklist.md`
3. `vc-pitchdeck-framing.md`
4. `vc-pitchdeck-framing-example-prism.md`
5. `vc-pitchdeck-investor-language.md`
6. `vc-pitchdeck-slide-patterns.md`
7. `vc-pitchdeck-redflags.md`
8. `vc-pitchdeck-speaker-notes.md`

### Example usage
- Rewrite an existing deck from source docs
- Pressure-test whether a draft feels descriptive or investable
- Generate speaker notes for a sparse Apple-style deck
- Audit a deck for red flags before sending to investors

If more pitch-deck companions are added later, keep the `vc-pitchdeck-` prefix so they stay grouped naturally in folder listings.
